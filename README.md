# nang.dev — Personal Portfolio Website

> Personal portfolio website built from scratch with vanilla HTML, CSS, and JavaScript.  
> Live at: **[ananglesmana.my.id](https://ananglesmana.my.id)**

---

## About

This is the source code of my personal portfolio site — a space to showcase who I am, what I can do, and what I've been building. No frameworks, no templates. Just raw HTML/CSS/JS with a lot of trial and error.

The site covers:
- **About** — background & transition from operations to data
- **Skills** — Python ecosystem, SQL, data tools
- **Projects** — hands-on projects I've built or contributed to
- **Journal** — notes, thoughts, and things I'm learning
- **Contact** — reach me if you want to collaborate

---

## Tech Stack

| Layer | Details |
|---|---|
| Markup | HTML5 |
| Styling | Vanilla CSS (custom properties, animations, responsive) |
| Scripting | Vanilla JavaScript |
| Font | Inter + JetBrains Mono (Google Fonts) |
| Hosting | Custom domain via [ananglesmana.my.id](https://ananglesmana.my.id) |

No build tools. No npm. No framework. Just files.

---

## Features

- Dark mode aesthetic with noise texture overlay
- Custom cursor (desktop only, auto-disabled on touch devices)
- Scroll-triggered reveal animations
- Animated skill bars
- Fully responsive — works on mobile & desktop
- Hamburger nav for mobile

---

## Run Locally

No setup needed. Just clone and open:

```bash
git clone https://github.com/bushidoways/porto.git
cd porto
# Open index.html in your browser
```

Or use Live Server extension in VS Code for auto-reload.
